Waarom een Smooth Scroller?

Ik kreeg de preloader die ik voorheen aan het maken was niet volledig werkend. Ik ben na gaan denken over andere functionaliteiten die ik vaak ik websites gebruik. Ik kwam al snel bij een Smooth scroll uit. Ik heb vorig jaar van Kevin een script gekregen die hier voor zorgde. Echter was dit standaard en kon ik niets aanpassen binnen deze code. Daarom heb ik besloten om een smooth scroll te maken waarin ik verschillende variabelen zelf kan aanpassen.

Ik gebruik dit niet alleen voor op mijn eigen portfolio website maar ook vaak voor websites die ik tijdens projecten of voor klanten maak. Ik kan dit framework nu ook in de toekomst blijven gebruiken. Het is gemakkelijk dat ik verschillende variabelen aan kan passen zoals de snelheid en de spacing rond om de div. Het aanpassen van deze variabelen is voor mij een handigheid omdat ik zo niet in de knoei kom met verschillende designs
